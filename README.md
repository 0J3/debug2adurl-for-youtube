# Debug2AdURL

Allows you to copy youtube debug information to get an ad's url

## Installation

```yml
yarn global add debug2adurl
```

## Usage

### Running one-time

```bash
debug2adurl
# OR
yt_debug2adurl
```

### Watching the Clipboard

```bash
debug2adurl -w
# OR
debug2adurl --watch
# OR
yt_debug2adurl -w
# OR
yt_debug2adurl --watch
```
